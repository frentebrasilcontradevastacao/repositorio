# Observatório da Devastação

Este diretório reunirá iniciativas voltadas ao monitoramento contínuo dos processos de degradação ambiental no Brasil.

## Objetivos

- Produzir análises periódicas;
- Integrar diferentes fontes de dados;
- Desenvolver metodologias de acompanhamento;
- Divulgar relatórios públicos.

## Eixos de atuação

- Desmatamento;
- Queimadas;
- Crimes ambientais;
- Pressões territoriais;
- Tendências socioambientais.
