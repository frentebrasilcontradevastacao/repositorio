# Políticas Públicas

Este diretório será dedicado ao desenvolvimento colaborativo de propostas para a proteção ambiental e a promoção do desenvolvimento sustentável.

## Objetivos

- Elaborar propostas legislativas;
- Construir cartas-compromisso;
- Produzir recomendações técnicas;
- Incentivar o diálogo suprapartidário.

## Áreas prioritárias

- Clima;
- Biodiversidade;
- Justiça socioambiental;
- Economia sustentável;
- Povos indígenas e comunidades tradicionais.

## Status

🚧 Em desenvolvimento.
