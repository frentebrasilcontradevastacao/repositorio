# Voluntariado

Este diretório apoiará a gestão e integração de pessoas interessadas em contribuir com a Frente Brasil Contra a Devastação.

## Objetivos

- Facilitar o acolhimento de novos colaboradores;
- Organizar grupos de trabalho;
- Reconhecer contribuições voluntárias;
- Fortalecer a participação cidadã.

## Possíveis áreas de atuação

- Pesquisa;
- Comunicação;
- Design;
- Tecnologia;
- Educação;
- Mobilização territorial.

## Status

🚧 Estrutura em desenvolvimento.
