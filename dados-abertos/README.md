# Dados Abertos

Este diretório reunirá bases de dados socioambientais produzidas, organizadas ou curadas pela Frente Brasil Contra a Devastação.

## Objetivos

- Promover transparência;
- Incentivar pesquisas independentes;
- Apoiar o jornalismo de dados;
- Subsidiar políticas públicas baseadas em evidências.

## Temas previstos

- Desmatamento;
- Queimadas;
- Biodiversidade;
- Recursos hídricos;
- Mudanças climáticas;
- Indicadores socioambientais.
