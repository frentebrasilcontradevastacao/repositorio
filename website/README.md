# Website

Este diretório será destinado ao desenvolvimento e manutenção do site institucional da Frente Brasil Contra a Devastação.

## Objetivos

- Disponibilizar informações institucionais;
- Divulgar campanhas e mobilizações;
- Publicar notícias e conteúdos educativos;
- Facilitar a participação cidadã;
- Integrar ferramentas digitais e plataformas colaborativas.

## Conteúdos previstos

- Código-fonte do site;
- Componentes visuais;
- Documentação técnica;
- Recursos multimídia;
- Integrações futuras.
