# Como contribuir

Existem diversas formas de colaboração:

## Pesquisa

- Revisão bibliográfica;
- Produção de relatórios;
- Curadoria de dados.

## Comunicação

- Design;
- Jornalismo;
- Produção audiovisual;
- Redes sociais.

## Tecnologia

- Desenvolvimento web;
- Ciência de dados;
- Visualização de dados;
- IA aplicada ao interesse público.

## Participação cidadã

- Organização territorial;
- Mobilização;
- Eventos locais.

Antes de contribuir, leia o Código de Conduta.
